5. **What is a Kaizen event?**  
   **Answer:** A Kaizen event is a focused, short-term improvement project to solve a specific problem.

6. **What is a Control Chart?**  
   **Answer:** A Control Chart monitors process variation over time and detects if it is stable or needs correction.

7. **What is a Green Belt?**  
   **Answer:** A Green Belt is a professional trained to lead process improvement projects using Lean Six Sigma tools.

8. **Difference between Lean and Six Sigma?**  
   **Answer:** Lean focuses on speed and waste reduction; Six Sigma focuses on reducing variation and defects.

## ðŸŸ¡ Intermediate Level

1. **What is a Value Stream Map (VSM)?**  
   **Answer:** A visual tool showing all steps in a process, highlighting waste and improvement opportunities.

2. **What is Poka-Yoke?**  
   **Answer:** Error-proofing technique to prevent mistakes in a process.

3. **Difference between CTQ and KPI?**  
   **Answer:** CTQ (Critical to Quality) is customer-focused requirement; KPI (Key Performance Indicator) measures performance.

4. **What is process sigma level?**  
   **Answer:** It indicates the performance of a process by the number of defects per million opportunities (DPMO).

5. **What is a Pareto Chart?**  
   **Answer:** A chart that identifies the most significant factors in a dataset using the 80/20 rule.

6. **Difference between qualitative and quantitative analysis?**  
   **Answer:** Qualitative analysis uses descriptive data; quantitative analysis uses numerical data and statistics.

7. **What is hypothesis testing in Six Sigma?**  
   **Answer:** A method to determine if there is a significant difference or relationship between data sets.

8. **What is Regression Analysis?**  
   **Answer:** A statistical tool to find relationships between variables and predict outcomes.

9. **Difference between process control and process improvement?**  
   **Answer:** Process control ensures stability; process improvement increases efficiency and reduces defects.

10. **What is a Kaizen Burndown Chart?**  
    **Answer:** A visual tool to track improvement activities and completion during a Kaizen event.

   **Answer:** Single-factor tests one variable at a time; multi-factor tests multiple variables simultaneously.

6. **Difference between Type I and Type II error?**  
   **Answer:** Type I rejects a true hypothesis; Type II fails to reject a false hypothesis.

7. **What is a hypothesis in Six Sigma?**  
   **Answer:** An assumption about the process that can be tested using data.

8. **How do you identify critical Xs in a process?**  
   **Answer:** Using tools like Fishbone diagram, Pareto chart, regression, and DOE analysis.

9. **What is Lean Six Sigma project charter?**  
   **Answer:** A document defining project scope, objectives, team roles, and expected benefits.

10. **What is the difference between Green Belt and Black Belt roles?**  
    **Answer:** Green Belts lead projects part-time; Black Belts lead larger projects full-time and mentor Green Belts.
